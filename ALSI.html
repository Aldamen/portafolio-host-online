ALSI SYSTEM CONCESIONARIO

Es un aplicativo de escritorio el cual puede administrar un empresa de venta de automoviles o concesionario para ingresar y almacenar información del comprador, del automovil e información relacionada a la venta.

Se utilizaron ciclos de vida basadas en Ingeniería de Software, Además de la implementación de una base de datos la cual guarda información relacionada para poder analizar los datos.

La aplicación tiene las siguientes caracteristicas:

  • Ingreso de información de Clientes, Vendedores y Automoviles
  • Creación de nuevos usuarios
  • Creación de contratos de pago con el cliente
  • Exportar la información en formato XLS (MS Excel)

Para el desarrollo e implementación se usaron los siguientes elementos:

  • Desarrollado en Lenguaje Java 8
  • Basados en la metodologia de desarrollo RUP
  • Se utilizaron las librerias: jcalendar, xmlbeans y RSS
  • Uso de la API POI con la cual se pueden leer y escribir ficheros MS Excel utilizando Java puro
  • Para el diseño y creación de la bases de datos se uso lenguaje SQL (Bases de Datos Relacionales)
