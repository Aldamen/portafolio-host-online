NETWORK-ATTACHED STORAGE SYSTEM FOR SMALL BUSINESS OR HOMENN

La virtualidad ha traído grandes cambios al mundo haciendo que aspectos como la educación, la cultura y la comunicación tomen un nuevo significado; con ello el manejo y gestión de datos se ha convertido en un tema importante para casi cualquier persona y aún más para aquellas que se dedican a la comercialización de productos y que quieren expandir su alcance por medio del internet.

Se trata de un sistema distribuido, implementando configuración de servidores mediante un full-stack framework y API Restful que facilita el acceso y la administración de datos empresariales.

  • Node.js para la elaboración del backend
  • Express.js como framework de Node
  • React.js para la elaboración del frontend
  • JSON web token para la transmision de datos
  • Uso de MongoDB para la base de datos
  • Moongose como libreria adicional de Node.js
  • Uso de GPL para licencia de software libre

La aplicación tiene las siguientes caracteristicas:

  • Ingreso de información de Usuarios, Archivos, Fotos, Videos y Bibliotecas
  • Creación de nuevos usuarios mediante uso de administrador
  • Sincronización con las carpetas de Windows y Ubuntu
  • Asignación de espacio de almacenamiento para cada usuario
