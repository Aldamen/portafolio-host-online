EASY AIRPORT MANAGER

Es un aplicativo web para la administración de boletos de avión y de aerolineas, el objetivo del aplicativo es el de facilitar el acceso a la información relacionada con viajes de avión, pasajeros y aerolineas.

La aplicación tiene las siguientes caracteristicas:

  • Ingreso de información de Viajes, costos y Aerolineas
  • Creación de nuevos usuarios
  • Integración con bases de datos (MySQL)

Para el desarrollo e implementación se usaron los siguientes elementos:

  • Desarrollado en Lenguaje Java 8
  • Basados en la metodologia de desarrollo Waterfall
  • Se utilizaron las librerias: Hibernate, Javax.faces y Primefaces
  • Para la implementación del servidor web se hizo uso de Apache Tomcat
  • Para el diseño y creación de la bases de datos se uso lenguaje MySQL (Bases de Datos Relacionales)
